# Sanity Studio

This folder contains a minimal Sanity Studio configuration for your blog content.

## Setup

1. Install dependencies inside `studio`:

```bash
cd studio
npm install
```

2. Run the studio locally:

```bash
npm run dev
```

3. Set the environment variables in the root `.env` file:

- `NEXT_PUBLIC_SANITY_PROJECT_ID`
- `NEXT_PUBLIC_SANITY_DATASET`
- `SANITY_API_TOKEN`
